# TechNews - Portal de Notícias

Portal de notícias sobre tecnologia, inovação e tendências do mercado tech.

## Estrutura do Projeto

```
projeto-noticias/
├── index.html          # Página principal
├── assets/
│   ├── icons/          # Ícones do projeto
│   ├── images/         # Imagens e banners
│   └── Logo.svg        # Logo do site
└── styles/
    ├── global.css      # Estilos globais
    ├── header.css     # Estilos do cabeçalho
    ├── index.css      # Estilos da página inicial
    ├── sections.css   # Estilos das seções
    └── utility.css    # Classes utilitárias
```

## Tecnologias

- **HTML5** - Estrutura semântica
- **CSS3** - Estilização com CSS Grid e Flexbox
- **Google Fonts** - Fonte Archivo

## Como Executar

1. Clone o repositório ou baixe os arquivos
2. Abra o arquivo `index.html` em seu navegador

```bash
# Via terminal (Windows)
start index.html
```

## Funcionalidades

- Layout responsivo com CSS Grid
- Menu de navegação com categorias tech
- Cards de notícias em destaque
- Design moderno com tipografia customizada
